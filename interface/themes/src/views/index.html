Overview

The OpenEMR-interface uses the Fabricator UI toolkit to document and standardize the creation of user interface elements. The project is using bootstrap as base. It contains different materials that can be assembled into more complex page layouts.

This guide contains real working examples, code snippets, documentation, and style guidelines.

Themes

There are currently 3 different theme categories in OpenEMR. The light theme is the default modern theme. The manila theme is a combination of OpenEMR's legacy themes (which have all been removed) with some modern elements. All the other "flat" themes are basically the same theme with different color palettes.

You can view any page in this guide in one of these themes using the dropdown in the upper right. Checking the "right-to-left" box will use the rtl variant of the theme
